📘 Fundamentos Matemáticos — Hidráulica con EDO

1. Ecuación de Continuidad (Conservación de Masa)

El principio fundamental que rige el comportamiento del nivel en un tanque es el balance de masa. Para un fluido incompresible:

$$A\frac{dh}{dt} = Q_{entrada}(t) - Q_{salida}(t)$$

donde $A$ es el área de la sección transversal (m²), $h(t)$ la altura del líquido (m) y $Q$ los caudales volumétricos (m³/s).

2. Ley de Torricelli — Descarga por Gravedad

La velocidad de eflujo a través de un orificio en la base del tanque, a una altura $h$, se obtiene de la conservación de energía de Bernoulli:

$$v = \sqrt{2gh}$$

Incorporando el coeficiente de descarga $C_d$ y el área del orificio $A_o$, se define la constante de descarga $k = C_d A_o \sqrt{2g}$, de modo que:

$$Q_{salida} = k\sqrt{h(t)}$$

3. EDO del Tanque Simple (Vaciado)

Forma general:

$$A\frac{dh}{dt} = -k\sqrt{h}$$

Esta es una EDO de primer orden no lineal (raíz cuadrada en $h$). Se resuelve por separación de variables:

$$\int h^{-1/2}\,dh = -\frac{k}{A}\int dt \;\Longrightarrow\; 2\sqrt{h} = -\frac{k}{A}t + C$$ $$\boxed{h(t) = \left(\sqrt{h_0} - \frac{k}{2A}\,t\right)^{\!2}, \qquad 0 \le t \le t_{vac}}$$

El tiempo de vaciado completo ($h \to 0$) es finito:

$$t_{vac} = \frac{2A\sqrt{h_0}}{k}$$

El caudal de salida varía con el tiempo como:

$$Q(t) = k\sqrt{h(t)} = k\left(\sqrt{h_0} - \frac{k}{2A}t\right)$$

4. Modelo con Bomba — EDO con Forzamiento Constante

Cuando existe un caudal de entrada $Q_{in}$ (constante), la EDO se convierte en una ecuación con término forzado:

$$A\frac{dh}{dt} = Q_{in} - k\sqrt{h}$$

El estado estacionario ($\dot{h}=0$) se alcanza cuando $Q_{in} = k\sqrt{h_{ss}}$, por tanto:

$$h_{ss} = \left(\frac{Q_{in}}{k}\right)^2$$

La respuesta transitoria linealizada alrededor de $h_{ss}$ tiene la forma $h'(t) = h'_0 e^{-t/\tau}$ con constante de tiempo:

$$\tau = \frac{2A\sqrt{h_{ss}}}{k}$$

5. Sistema de Doble Tanque — Sistema de EDOs Acopladas

Para dos tanques en serie, se plantea un sistema de dos ecuaciones diferenciales ordinarias de primer orden acopladas:

$$A_1\frac{dh_1}{dt} = Q_{in} - k_{12}\sqrt{\max(h_1-h_2,\,0)}$$ $$A_2\frac{dh_2}{dt} = k_{12}\sqrt{\max(h_1-h_2,\,0)} - k\sqrt{h_2}$$

Este sistema no admite solución analítica cerrada general y se resuelve numéricamente.

6. Método Numérico — Runge-Kutta de 4° Orden (RK4)

Para resolver $\dot{y} = f(t,y)$ con condición inicial $y(t_0)=y_0$, el método RK4 aproxima la solución con error de truncamiento local $\mathcal{O}(\Delta t^5)$:

$$k_1 = f(t_n,\,y_n)$$ $$k_2 = f\!\left(t_n+\tfrac{\Delta t}{2},\;y_n+\tfrac{\Delta t}{2}k_1\right)$$ $$k_3 = f\!\left(t_n+\tfrac{\Delta t}{2},\;y_n+\tfrac{\Delta t}{2}k_2\right)$$ $$k_4 = f(t_n+\Delta t,\;y_n+\Delta t\,k_3)$$ $$\boxed{y_{n+1} = y_n + \frac{\Delta t}{6}(k_1 + 2k_2 + 2k_3 + k_4)}$$

7. Energía y Potencia Hidráulica

La energía potencial almacenada en el fluido ($\rho = 1000$ kg/m³, $g = 9.81$ m/s²):

$$E(t) = \frac{1}{2}\rho g A\,h(t)^2 \qquad [J]$$

La potencia hidráulica instantánea de salida:

$$P(t) = \rho g\,Q(t)\,h(t) = \rho g\,k\,h^{3/2}(t) \qquad [W]$$

8. Clasificación del Comportamiento del Sistema

  • Vaciado puro: $Q_{in}=0$ → $h(t) \to 0$ en tiempo finito $t_{vac}$.
  • Equilibrio estable: $Q_{in} = k\sqrt{h_{ss}}$ → $h(t) \to h_{ss}$ (asintótico).
  • Llenado progresivo: $Q_{in} > k\sqrt{h_{max}}$ → desbordamiento si no hay control.
  • Respuesta transitoria: convergencia exponencial linealizada al punto de equilibrio.
